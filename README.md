# Gratitude

A gentle, accessible gratitude check-in page with adjustable text sizes, theme contrast, and reduced-motion support.

## Getting started
Open `index.html` in your browser. Preferences for text size, high-contrast theme, and reduced motion are saved locally.

## Accessibility features
- High-contrast toggle with clear focus outlines for keyboard navigation.
- Text size slider scales the entire page using CSS variables.
- Reduced-motion toggle removes animations/transitions for calmer visuals.
- Screen reader labels for all controls and compassionate, inclusive prompts.
- Skip link, logical heading order, and focus-visible styles for easy navigation.
